## Usage in Templates

### How to Add Images
Replace template image URLs with these premium Unsplash images.

Example:
```html
<img src="https://images.unsplash.com/photo-1576091160550-112173f7f869?w=400&q=80" alt="Medical">
```

## Image Optimization Tips
- Use `?w=` parameter to set width (recommended sizes: 200, 400, 800, 1200)
- Use `?q=` parameter to set quality (80-90 recommended)
- Always add descriptive `alt` text for accessibility
- Use responsive images with srcset for different screen sizes

## Recommended Image Sizes
- Hero Section: 1200x600px
- Department Cards: 300x200px
- Doctor Profiles: 400x500px
- Service Cards: 400x300px
- Background Images: 1920x1080px
